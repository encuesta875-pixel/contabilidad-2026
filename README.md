# 📊 ContaPro - Sistema de Contabilidad

Sistema completo de contabilidad desarrollado con Spring Boot para gestionar la contabilidad de pequeñas y medianas empresas.

## 🚀 Características Principales

- Plan de Cuentas Contables
- Asientos Contables
- Libro Diario y Libro Mayor
- Balance General
- Estado de Resultados
- Sistema de Licencias Opcional (7 días de prueba)
- Autenticación con Google OAuth2

## 🛠️ Tecnologías

- Spring Boot 3.4.5
- MySQL
- Thymeleaf
- Spring Security
- Maven

## 📋 Instalación

1. Clonar repositorio
2. Configurar MySQL
3. Copiar `application.properties.example` a `application.properties`
4. Ejecutar: `mvn spring-boot:run`
